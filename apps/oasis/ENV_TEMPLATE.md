# Environment Variables Template

Copy this template to `.env` file in the oasis app directory.

```bash
# Private key for resolver wallet (without 0x prefix or with)
PRIVATE_KEY=your_private_key_here

# RPC URLs
SEPOLIA_RPC_URL=https://sepolia.infura.io/v3/YOUR_INFURA_KEY
CELO_RPC_URL=https://forno.celo.org

# Contract Addresses
CONTRACT_ADDRESS_SEPOLIA=0x49F1b8A77712Edf77Fa5d04D07d77a846B23A91B
CONTRACT_ADDRESS_CELO=0x243E571194C89E8B848137EdB46e5A1156272860

# Aviation Edge API Key (for flight status)
AVIATION_EDGE_API_KEY=your_aviation_edge_api_key_here

# Server Port (optional, defaults to 4500)
PORT=4500
```
